### Przeznaczenie \ Purpose

**EN:** A hands-on PHP adaptation of the design patterns from the classic book "Head First: Design Patterns". This repository contains my implementations, notes, and practical examples translated from the original Java code. Each pattern (Strategy, Observer, Decorator, Factory, etc.) includes working PHP examples, explanations of its use case, and the benefits it provides. Perfect for PHP developers wanting to understand and apply these essential concepts.

**PL:** Praktyczna adaptacja wzorców projektowych z kultowej książki "Head First: Design Patterns" (wyd. pol. "Rusz głową! Wzorce Projektowe") na język PHP. Zawiera moje implementacje, notatki i przykłady przetłumaczone z oryginalnego kodu Java. Każdy wzorzec (Strategia, Obserwator, Dekorator, Fabryka, itp.) opatrzony jest działającym przykładem w PHP, wyjaśnieniem zastosowania i korzyści. Idealne dla programistów PHP chcących zrozumieć i stosować te kluczowe koncepcje.

[link](https://github.com/kamilszwaradzki/WzorceProjektowe)

## Koncepcja DDD

### Przeznaczenie \ Purpose

**EN:** A knowledge repository documenting my deep dive into Domain-Driven Design through Vladimir Khononov's book "Domain-Driven Design: Patterns, Principles, and Practices". Contains detailed chapter summaries, distilled concepts (Bounded Contexts, Aggregates, Domain Events, Ubiquitous Language), and personal reflections on applying DDD principles to complex software design. Serves as both a personal reference and a resource for others learning DDD.

**PL:** Repozytorium wiedzy dokumentujące moją pogłębioną naukę Domain-Driven Design poprzez książkę Vladimira Khononova "Domain-Driven Design: Wzorce, zasady i praktyki". Zawiera szczegółowe podsumowania rozdziałów, esencję kluczowych koncepcji (Bounded Contexts, Aggregates, Domain Events, Język Wszechobecny) oraz osobiste przemyślenia na temat zastosowania zasad DDD w projektowaniu złożonych systemów. Pełni rolę zarówno osobistego notatnika, jak i zasobu dla innych uczących się DDD.

[link](https://github.com/kamilszwaradzki/Koncepcja_DDD)
